# 🚀 Jair Rueda - Portafolio Personal

¡Bienvenido a mi portafolio profesional! Este proyecto es una aplicación web responsiva, rápida y moderna diseñada para destacar mi experiencia, habilidades y proyectos como Ingeniero de Software & Full-Stack Developer.

## 🌟 Características Principales

- **Diseño Premium & Glassmorphism:** Interfaz de usuario moderna con efectos translúcidos (glassmorphism) y un diseño minimalista.
- **Modo Oscuro/Claro (Dark Mode):** Inicia en modo claro por defecto, pero incluye un interruptor (switch) dinámico para que los usuarios puedan cambiar al modo oscuro. Todos los colores se adaptan perfectamente.
- **Multilenguaje (i18n):** Soporte completo e integrado para Español e Inglés sin necesidad de recargar la página. Modifica todo el contenido y los placeholders de los formularios en tiempo real.
- **Experiencia 3D Interactiva:** Integración de **Three.js** para renderizar un orbe de partículas tridimensional que reacciona de manera fluida a los movimientos del ratón, y cambia de color según el tema (claro/oscuro).
- **Formulario de Contacto Funcional:** Implementación con **EmailJS** para el envío directo de correos desde la web de forma segura.
- **Optimizado para el Rendimiento:** Desarrollado con **Astro** para entregar HTML estático con un peso mínimo, asegurando tiempos de carga ultrarrápidos y excelente SEO.

## 💻 Tecnologías Utilizadas

- **Framework Principal:** [Astro v5](https://astro.build/)
- **Estilos:** Vanilla CSS (CSS Variables para tematización)
- **Animaciones 3D:** [Three.js](https://threejs.org/)
- **Iconografía:** [Phosphor Icons](https://phosphoricons.com/) y [Devicon](https://devicon.dev/)
- **Gestión de Correos:** [EmailJS](https://www.emailjs.com/)

## 🛠️ Instalación y Configuración Local

Si deseas correr este proyecto localmente, sigue estos pasos:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/JairAlexey/MiPortafolio.git
   cd MiPortafolio
   ```

2. **Instala las dependencias:**
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
   Abre `http://localhost:4321` en tu navegador para ver la web.

4. **Construir para Producción:**
   ```bash
   npm run build
   ```
   Esto generará los archivos estáticos optimizados en la carpeta `./dist/`.
